# AFLHR Lite Documentation

Built with [Docusaurus](https://docusaurus.io/).

## Installation

```bash
npm install
```

## Local Development

```bash
npm start -- --port 4000
```

Or via the project root:

```bash
make start    # starts backend + frontend + docs
```

## Build

```bash
npm run build
```

Generates static files in `build/`.
